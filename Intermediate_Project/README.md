# Student Marks Analysis

## Description
This project analyzes student marks to understand performance trends and visualize data.

## Dataset
- Source: Self-created dataset
- Description: Contains marks of students in different subjects

## Steps Performed
1. Created dataset
2. Loaded data using Pandas
3. Performed basic analysis
4. Created visualizations

## Results
- Identified student performance trends
- Visualized subject-wise marks

## Tools Used
- Python
- Pandas
- Matplotlib

## Conclusion
This project shows how data analysis can be used to understand student performance.
